# 이도현 포트폴리오

프론트엔드 개발자 이도현의 개인 포트폴리오 페이지입니다.

- 배포 링크: https://dohyeonl.github.io/PORTFOLIO/

## 소개하는 프로젝트

- **AI-VIS** — 건설현장 안전장비 착용 여부를 CCTV 영상으로 실시간 감지하는 AI 안전관리 플랫폼 ([GitHub](https://github.com/projectTFIC/projectTFIC))
- **PODO** — 밤길 귀가가 불안한 노약자·여성·아동을 위한 안전 경로 안내 서비스 ([GitHub](https://github.com/DoHyeonL/PODO))
- **마녀공장 사이트 리디자인** — 마녀공장 공식 쇼핑몰을 레퍼런스로 삼아 1인으로 다시 설계하고 구현한 프로젝트 ([GitHub](https://github.com/DoHyeonL/manyo-redesign) · [배포 링크](https://dohyeonl.github.io/manyo-redesign/))

## 사용 기술

- HTML5 / CSS3
- Vanilla JavaScript
- Google Fonts, Devicon

## 실행 방법

1. 저장소를 다운로드 또는 clone
2. index.html을 브라우저로 열면 바로 확인 가능
